# Shadow Tail

一人称視点風の尾行ゲームです。

プレイヤーは前方を歩く人物を追跡します。近づきすぎると怪しまれ、離れすぎると見失います。対象が振り返ったときは、電信柱・ポスト・看板・自販機などの遮蔽物に隠れて発見を避けます。

## ゲームの目的

対象に見つからず、一定距離を尾行し続けるとミッションクリアです。

失敗条件は主に次の2つです。

- 対象が振り返っている間に姿を見られる
- 対象との距離管理に失敗し、警戒ゲージが最大になる

## 起動方法

依存関係はありません。ブラウザだけで動きます。

macOS では次のコマンドで起動できます。

```bash
open index.html
```

または、`index.html` をブラウザに直接ドラッグして開いてください。

## 操作

| キー | 動作 |
| --- | --- |
| `W` / `↑` | 対象に近づく |
| `S` / `↓` | 対象から距離を取る |
| `A` / `←` | 左へ移動 |
| `D` / `→` | 右へ移動 |
| `Space` | 近くの遮蔽物に隠れる |

## 画面の見方

- `DIST`: 対象との距離が適切かを示します
- `ALERT`: 警戒ゲージです。最大になると失敗です
- `TAILING`: 通常の尾行状態です
- `WARNING`: 対象が振り返る直前です
- `WATCH`: 対象が振り返っている状態です
- `HIDDEN`: 遮蔽物に隠れている状態です

## プレイのコツ

- 対象の近くに寄りすぎないようにしてください
- `WARNING` が出たら、近くの遮蔽物へ移動してください
- 遮蔽物の足元に黄色い輪が出たら、`Space` で隠れられます
- 対象がこちらを見ている間は、隠れたまま動かないのが安全です

## 実装済み

- 奥行きのある疑似3Dの二次元 Canvas 画面
- 前方を歩く尾行対象
- 振り返りイベントと予兆表示
- 電信柱、ポスト、看板、自販機の遮蔽物
- 遮蔽物付近での隠れ判定
- 距離メーター、警戒ゲージ、発見/成功判定
